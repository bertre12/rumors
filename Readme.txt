В данном проекте было реализовано:

•	Создание двух приложений с различным функционалом.
•	Создание и реализация модели, таблицы в базе данных.
•	Создание admin пользователя.
•	Реализовано два метода работы при вызове клиентских запросов (функция и класс).
•	Создание статических файлов (CSS), файлов-шаблонов (HTML).
•	Подключение сторонних стилей (bootstrap).
•	Создание формы для внесения данных.
•	Реализован метод добавления, редактирования и удаления данных.
•	Реализован метод динамических URL адресов.
